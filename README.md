Speed up your workflow with these default hotkeys (customizable in `chrome://extensions/shortcuts`):

| Shortcut | Action | Description |
|----------|--------|-------------|
| **`Ctrl + Shift + V`** | Open ClipTrack | Instantly open the extension popup. |

## üõ†Ô∏è Installation

1.  Clone this repository:
    ```bash
    git clone https://github.com/haider-subhan/ClipTrack---Clipboard-History-Manager.git
    ```
2.  Open Chrome and go to `chrome://extensions/`.
3.  Enable **Developer Mode** (top right switch).
4.  Click **Load unpacked**.
5.  Select the folder where you cloned this repository.
6.  Pin **ClipTrack** to your toolbar for easy access!
