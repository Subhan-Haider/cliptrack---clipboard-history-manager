# Quick Installation Guide

## Your extension is ready! Follow these steps:

### Step 1: Open Chrome Extensions Page
1. Open Google Chrome (or any Chromium-based browser like Edge, Brave)
2. Type `chrome://extensions/` in the address bar and press Enter
   - For Edge: `edge://extensions/`
   - For Brave: `brave://extensions/`

### Step 2: Enable Developer Mode
1. Look for a toggle switch labeled "Developer mode" in the top-right corner
2. Turn it ON

### Step 3: Load the Extension
1. Click the "Load unpacked" button (appears after enabling Developer mode)
2. Navigate to and select this folder: `C:\Users\setup\Videos\exten2`
3. Click "Select Folder"

### Step 4: Pin the Extension
1. Click the puzzle piece icon (ðŸ§©) in your browser toolbar
2. Find "ClipTrack - Clipboard History Manager"
3. Click the pin icon to keep it visible in your toolbar

### Step 5: Start Using!
1. Click the ClipTrack icon in your toolbar
2. Copy or paste any text on any webpage
3. Open ClipTrack to see your clipboard history!

## Troubleshooting:

**Extension won't load?**
- Make sure you selected the correct folder (exten2)
- All required files should be present

**History not tracking?**
- Make sure you've granted clipboard permissions
- Try refreshing the webpage you're copying from
- Check settings to ensure tracking is enabled

**Need help?**
- Check the README.md file for detailed documentation
- All data is stored locally - no internet required!
